# 数据操作汇总

本文档提供了巡检记录助手应用中所有数据交互的全面概览，以便于后续开发 RESTful API 后端。

## 1. 授权模块

| 触发元素 | 操作类型 | 数据描述 |
|---------|---------|---------|
| 登录表单 | 读取 (Read) | 验证用户凭据 |
| 登录按钮 | 创建 (Create) | 创建用户会话和认证令牌 |
| 登出按钮 | 删除 (Delete) | 删除用户会话和认证令牌 |

## 2. 巡检问题记录模块

### 问题记录页面 (app/issues/page.tsx)

| 触发元素 | 操作类型 | 数据描述 |
|---------|---------|---------|
| 页面加载 | 读取 (Read) | 从 mockIssues 数组获取问题记录列表 |
| 状态筛选下拉框 | 读取 (Read) | 基于状态(全部/待处理/已闭环)过滤问题记录 |
| 问题记录复选框 | 更新 (Update) | 选择/取消选择问题记录，更新 selectedIssues 数组 |
| "合并问题"按钮 | 创建/更新 (Create/Update) | 合并选中的问题记录 |
| "生成巡检记录"按钮 | 创建 (Create) | 基于选中的问题记录生成巡检记录 |
| "查看详情"按钮 | 读取 (Read) | 查看问题记录详情 |
